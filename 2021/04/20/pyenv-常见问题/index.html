pyenv 常见问题

Apr 20, 2021

在 pyenv 安装的 python 使用 pyinstaller 等工具可能会出现以下信息:

  • 解决方法: 重新用 pyenv 安装 python 时添加 CONFIGURE_OPTS 变量

    • linux: --enable-shared
    • macos: --enable-framework
    • 使用 sudo 是因为可能会出现安装时权限不够的情况
